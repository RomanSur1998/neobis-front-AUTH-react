# React + Vite

- Проект , для отработки запросов к API отвечающих за регистрацию и авторизаию пользователя в приложении .

Основной фукционал  
Регистрация :

- Ввод почты
- Ввод Имени пользователя
- Ввод пародля
- Подтверждение пароля

Подтверждение регистрации:

- Ввод кода , пришедщего на почту

Авторизация :

- Ввод почты
- Ввод пароля
